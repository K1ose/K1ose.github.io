tmux

tmux是一个terminal multiplexer（终端复用器），可以启动一系列终端会话， 它解绑了会话和终端窗口。关闭终端窗口再打开，会话并不终止，而是继续运行在执行。将会话与终端窗后彻底分离。 tmux使用C语言实现，可运行在OpenBSD，FreeBSD，NetBSD，Linux，OS，X，Solaris上。

帮助

会话（Session）管理

  • s 列出所有会话

  • $ 重命名当前的会话

  • d 断开当前的会话

窗口（Window）管理

  • c 创建一个新窗口

  • , 重命名当前窗口

  • w 列出所有窗口

  • % 水平分割窗口

  • “ 竖直分割窗口

  • n 选择下一个窗口

  • p 选择上一个窗口

  • 09 选择09对应的窗口

窗格（Pane）管理

  • % 创建一个水平窗格

  • “ 创建一个竖直窗格

  • q 显示窗格的编号

  • o 在窗格间切换

  • } 与下一个窗格交换位置

  • { 与上一个窗格交换位置

  • ! 在新窗口中显示当前窗格

  • x 关闭当前窗格

其他

  • t 在当前窗格显示时间
