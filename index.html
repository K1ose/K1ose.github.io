Introduction 简介 Docker 最初是由 dotCloud 公司创始人 Solomon Hykes 在法国期间发起的一个公司内部项目。其使用 Go 语言进行开发，基于 Linux 内核的 cgroup 、 namespace 、 AUFS 类的 Union FS 等技术，对进程进行封装隔离，属于操作系统层面的虚拟化技术。 由于隔离的进程独立于宿主和其他的隔离进程，因此也称其为容器。最初实现基于 LXC ，从 0.7 版本后开始去除 LXC ，转而使用自行研发的 libcontainer ，从 1.11 版本开始则使用 runC 和 containerd 。 Docker与传统的虚拟化方式不同在于： 传统虚拟机技术是虚拟出一套硬件后，在器上运行一个完整操作系统，再再该系统上运行所需应用进程。 容器内的应用进程直接运行于宿主的内核，容器内没有自己的内核，也没有虚拟的硬件。 这也突出了 Docker 的优点： 更高效的利用系统资源：不需要虚拟硬件以及运行完整的操作系统。 更快速的启动时间：直接运行与宿主内核，无需启动完整的操作系统。 抑制的运行环境：Docker的镜像提供 ...
